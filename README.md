# 综合性电商网站

## 项目简介

这是一个使用纯HTML5、CSS3和TypeScript开发的综合性电商网站项目，实现了完整的前端功能模块和用户交互体验。

## 技术栈

- **HTML5**: 语义化标签、无障碍访问(ARIA)
- **CSS3**: Flexbox/Grid布局、CSS变量、关键帧动画、响应式设计
- **TypeScript**: ES6+语法、类型安全、模块化开发
- **本地存储**: localStorage实现数据持久化

## 核心功能

### 1. 商品展示系统
- ✅ 商品列表页(网格布局)
- ✅ 分类筛选(电子产品、时尚服饰、家居生活、图书文具)
- ✅ 商品详情页(图片轮播、规格展示)
- ✅ 搜索功能(关键词高亮)
- ✅ 商品排序(价格、销量、评分)

### 2. 购物车管理
- ✅ 添加/删除商品
- ✅ 修改商品数量
- ✅ 计算总价(含运费、折扣)
- ✅ localStorage持久化

### 3. 用户交互系统
- ✅ 表单验证(登录表单)
- ✅ 弹窗组件(商品详情、登录)
- ✅ Toast通知
- ✅ 加载动画(骨架屏、页面加载)

### 4. 响应式设计
- ✅ 适配手机、平板、桌面端
- ✅ 移动端汉堡菜单
- ✅ 3个断点(480px、768px、1200px)

### 5. 高级功能
- ✅ 暗黑模式切换
- ✅ 最近浏览历史
- ✅ 图片懒加载
- ✅ 搜索防抖优化
- ✅ 微交互动画

## 性能优化

- 图片懒加载(Intersection Observer API)
- 防抖/节流(搜索输入优化)
- CSS动画硬件加速
- 事件委托减少监听器数量

## 安装步骤

1. **安装依赖**
```bash
npm install
```

2. **编译TypeScript**
```bash
npm run build
```

3. **开发模式(自动编译)**
```bash
npm run watch
```

4. **启动本地服务器**

使用任意HTTP服务器工具，例如:
- Live Server (VS Code插件)
- Python: `python -m http.server 8000`
- Node.js: `npx serve`

## 主要特性说明

### 主题系统
使用CSS变量实现完整的主题切换系统，支持亮色和暗黑模式。主题选择会保存在localStorage中。

### 购物车
- 侧边栏设计，不打断浏览体验
- 实时计算小计、运费、总价
- 满99元免运费
- 数据持久化到localStorage

### 搜索功能
- 实时搜索建议
- 关键词高亮显示
- 300ms防抖优化性能

### 商品详情
- 模态弹窗展示
- 图片轮播功能
- 规格参数展示
- 添加到浏览历史

### 表单验证
- 实时验证反馈
- 错误提示
- 邮箱格式验证
- 密码长度验证

## 技术亮点

1. **TypeScript类型安全**: 完整的类型定义和接口声明
2. **模块化架构**: Store/UI/Utils分离，职责清晰
3. **响应式设计**: 移动优先，3个断点适配
4. **性能优化**: 懒加载、防抖、节流、事件委托
5. **用户体验**: 骨架屏、加载动画、Toast通知、微交互
6. **无障碍访问**: 语义化HTML、ARIA属性、键盘导航
